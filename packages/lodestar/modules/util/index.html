    For each byte check if a includes b, | a | b | result | | -------- | -------- | ------------- | | 00001111 | 00001111 | A equals B | | 00001111 | 00000011 | A superset B | | 00000011 | 00001111 | A subset B | | 11110000 | 00001111 | A exclude B | | 11111100 | 00111111 | A intersect B |

    For all bytes in BitArray: - equals = MAYBE ONLY equals - excludes = MUST ONLY equals - superset = MUST superset MAYBE equal - subset = MUST subset MAYBE equal - intersect = any other condition
